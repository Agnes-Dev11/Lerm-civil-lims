# LERM-CIVIL Lite

Soil & Material Testing Lab Management System built using Spring Boot.

## Features
- JWT Authentication
- Spring Security
- REST APIs
- CRUD Operations
- PostgreSQL
- Docker Support

## Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- Hibernate / JPA
- Docker

## Run
```bash
mvn spring-boot:run
```

# Lerm-civil-lims
LERM-CIVIL Lite is a Spring Boot–based Laboratory Information Management System (LIMS) inspired by real-world ERP workflows used in soil and civil testing laboratories. Built using Java, Spring Boot, PostgreSQL, REST APIs, and Docker with layered backend architecture.
